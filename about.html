ℹ️ About LuckyWheel Pro

A free, powerful event tool for lucky draws, prize reveals and surprise boxes — built for schools, offices and festivals worldwide.

What is LuckyWheel Pro?

LuckyWheel Pro is a free, browser-based event tool that combines three powerful features into one single-file application: a classic Spin Wheel for picking random winners, a QR Prize Code system for hidden prize reveals via smartphone scan, and a Lucky Box grid for opening surprise gift boxes at live events.

It was built to be completely serverless — all data lives in your browser, nothing is sent to any server, and it works fully offline once loaded. No account, no subscription, no ads.

🎡
Spin Wheel
Pick a random winner from up to 1,000 participants with beautiful festival themes
📲
QR Prize Code
Hidden prize reveals via QR scan with auto/manual timer and scan deduplication
🎁
Lucky Box
Grid of surprise gift boxes — click to open and reveal a random prize with confetti

Who is it for?

LuckyWheel Pro is designed for anyone who needs to pick a winner fairly and engagingly:

  • 🏫 Schools & Colleges — classroom activities, prize-giving ceremonies, student competitions
  • 🏢 Corporate Events — team building, year-end dinners, annual galas, conference giveaways
  • 🎂 Family & Social Functions — birthday parties, weddings, family reunions
  • 🪔 Festival Celebrations — Diwali, Chinese New Year, Hari Raya, Deepavali, Christmas, Holi, Halloween
  • 🌍 Global Events — 14 festival themes covering cultures from Malaysia, India, China, and worldwide

Technology

LuckyWheel Pro is built with pure web technologies — no frameworks, no build steps, no dependencies except the QR code library. It runs entirely in the browser.

HTML5 CSS3 Custom Properties Vanilla JavaScript (ES6+) Canvas API Web Audio API localStorage QRCode.js Google Fonts URLSearchParams requestAnimationFrame
